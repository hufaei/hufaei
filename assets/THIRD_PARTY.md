# Third-party assets and services

- [Skill Icons](https://github.com/tandpfun/skill-icons) — MIT License. The generated icon strip is stored locally as `skill-icons.svg`.
- [GitHub Profile Views Counter](https://github.com/antonkomarev/github-profile-views-counter) — MIT License. README embeds its hosted counter endpoint.
- [snk](https://github.com/Platane/snk) — GitHub Action used to generate the contribution snake.

`worldline-stats.svg` is generated by this repository's workflow from the official GitHub REST API; it does not rely on a third-party statistics renderer.
